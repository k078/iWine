Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda.

iWine

Deze applicatie biedt de gebruiker de optie om een wijnkelder bij te houden en te starten. In deze wijnkelder kunnen wijnflessen worden toegevoegd en worden bekeken. Een gebruiker kan ook meerdere wijnkelders aanmaken en beheren. Je kan dus een eigen wijncollectie bijhouden.

Entiteiten

Hier geef je een toelichting op je entiteiten. Voeg ook een afbeelding van je model toe zoals dat bij NoSql gevraagd wordt.

Gebruiker

  • id: number
  • gebruikersname: string
  • wachtwoord: string
  • email: string
  • totaalwaarde: number
  • cellarList:Cellar[]
  • vriendenlijst:User[]

Kelder

  • id: number
  • adres: string
  • eigenaar: string | User
  • oppervlakte: number
  • wijnflessenLijst: Winebottle[]
  • totaalwaarde:number

Wijnfles

  • id: number
  • wijnsoort: WineType
  • naam: string
  • jaar: number
  • land:string
  • prijs:number
  • zelfGemaakt:boolean
  • wijngaard:Winery
  • kelderLijst:Cellar[]
  • img:string

Wijngaard

  • naam: string

Userstories

Hier verwoord je in de vorm van userstories de functionaliteit van jouw applicatie.

Nr Userstory Evt toelichting
1 Als gebruiker wil ik een overzicht van wijnflessen kunnen bekijken. Ik wil in een scherm de inhoud van mijn wijnkelder kunnen bekijken.
2 Als ingelogde gebruiker wil ik wijnflessen verwijderen uit mijn kelder. Er moet een mogelijkheid zijn om een fles te kunnen verwijderen uit de kelder.
3 Als ingelogde gebruiker wil ik wijnflessen kunnen toevoegen aan mijn kelder. Bij een wijnfles zal een knop staan waarmee ik deze entiteit kan toevoegen aan mijn eigen wijnkelder.
4

Als ingelogde gebruiker wil ik zelf een wijnkelder aan kunnen maken.

 Een invoerscherm tonen waarmee het aanmaken van een kelder mogelijk is.
5

Als ingelogde gebruiker wil ik informatie van een wijnfles kunnen raadplegen.

 Informatiescherm openen van de gekozen wijnfles met informatie.
